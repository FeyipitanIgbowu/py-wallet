# FeyiPay — py-wallet

A personal wallet management application built with Python and Django.
FeyiPay allows users to manage their finances — deposit, withdraw, 
track balances, and view transaction history.

---

## Features

- Deposit and withdraw funds
- Real-time wallet balance
- Transaction history
- Secure data handling
- Unit tested with Django's test suite

---

## Tech Stack

- Python
- Django
- SQLite
- uv (dependency management)
